Help & Documentation

A guide to using Preprint Bot to track arXiv research.

Getting Started

1. Sign in or Create an Account

Use your email and password to sign in. If you are new, click Create Account on the login screen.

2. Create a Profile

Navigate to Profiles → + New Profile. Each profile represents a research interest area. Fill out:

  • Name – e.g. "AI Research"
  • arXiv Categories – the sub-fields to track (at least one required)
  • Threshold – how strict the matching should be (lower = more results)
  • Keywords – optional extra search terms

3. Upload Your Papers

In the profile list, expand the Papers section to upload PDFs or add papers by arXiv ID. These papers teach the system your research taste.

4. View Recommendations

After the daily pipeline runs, check the Dashboard for the latest matches or use the Recommendations tab to filter and search across all dates.

Page Guide

Dashboard

Displays your profile/corpus counts and the most recent day's recommendations across all profiles.

Profiles

Create, edit, and delete profiles. Upload PDFs or add arXiv papers directly. Each profile has its own frequency, threshold, and category settings.

Recommendations

Browse the full history of recommended papers. Filter by profile, date range, minimum score, keyword search, or specific arXiv categories.

Settings

Update your display name and email. View your account ID and creation date.

Troubleshooting

No recommendations showing?

  • Make sure you have uploaded at least a few papers to a profile.
  • The pipeline runs daily after arXiv publishes (around 9 PM EST). New profiles may take a day.
  • Try lowering your threshold – a very high value will filter out most papers.
  • Check that you selected active categories (e.g. cs.LG receives many daily submissions).

Contact

If something looks wrong, email preprintbot@syr.edu with details about the issue.
